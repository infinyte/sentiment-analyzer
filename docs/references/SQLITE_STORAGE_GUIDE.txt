SENTIMENT ANALYZER - SQLITE STORAGE IMPLEMENTATION
==================================================

OVERVIEW
--------

SQLite provides a lightweight, file-based database that requires NO external 
services. Perfect for:
- Local development
- Testing
- Single-server deployment
- Offline-first applications

Single file database: sentiment_analyzer.db (< 100MB)

---

WHEN TO USE SQLITE VS MONGODB
==============================

Use SQLite if:
✅ You want zero setup (file-based)
✅ You're working offline
✅ You prefer SQL queries
✅ File size < 100MB is fine
✅ You don't need cloud sync yet
✅ Single-server deployment

Use MongoDB if:
✅ You want cloud-hosted option (Atlas)
✅ You plan to scale horizontally
✅ You prefer document queries
✅ You need automatic replication
✅ You want smooth Azure migration

---

INSTALLATION
=============

1. Install sqlite3 npm package:
   npm install sqlite3 better-sqlite3

2. No server needed - database is a file!

3. That's it! Database auto-created on first use.

CREATE TABLE IF NOT EXISTS coins (

  symbol TEXT UNIQUE NOT NULL,
  name TEXT NOT NULL,
  price_usd REAL NOT NULL,
  market_cap_usd REAL,
  volume_24h_usd REAL,
  price_change_24h_percent REAL,
  price_change_7d_percent REAL,
  volatility_24h REAL,
  volatility_7d REAL,
  sentiment_score TEXT,
  sentiment_confidence REAL,
  sentiment_summary TEXT,
  market_rank INTEGER,
  trending_score INTEGER,
  headline_sample TEXT, -- JSON string
  timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
  source TEXT,
  updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- Coin snapshots (historical - every 10 minutes)
CREATE TABLE IF NOT EXISTS coin_snapshots (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  symbol TEXT NOT NULL,
  price_usd REAL NOT NULL,
  volume_24h_usd REAL,
  sentiment_score TEXT,
  sentiment_confidence REAL,
  timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
  FOREIGN KEY(symbol) REFERENCES coins(symbol)
);

-- Sentiment analysis cache (daily)
CREATE TABLE IF NOT EXISTS sentiment_cache (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  symbol TEXT NOT NULL,
  analysis_date TEXT NOT NULL UNIQUE,
  sentiment_score TEXT NOT NULL,
  confidence REAL,
  summary TEXT,
  key_catalysts TEXT, -- JSON array
  risk_factors TEXT,  -- JSON array
  short_term_outlook TEXT,
  volatility_warning BOOLEAN,
  generated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
  model TEXT,
  tokens_used INTEGER,
  expires_at DATETIME, -- TTL: 24 hours from generated_at
  FOREIGN KEY(symbol) REFERENCES coins(symbol),
  UNIQUE(symbol, analysis_date)
);

-- Price history (OHLCV data)
CREATE TABLE IF NOT EXISTS price_history (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  symbol TEXT NOT NULL,
  date TEXT NOT NULL,
  open REAL,
  high REAL,
  low REAL,
  close REAL,
  volume REAL,
  timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,

BACKEND IMPLEMENTATION
======================

File: backend/src/storage.ts

import Database from 'better-sqlite3';
import path from 'path';
import fs from 'fs';

interface StorageService {
  connect(): Promise<void>;
  disconnect(): Promise<void>;
  
  saveCoinSnapshot(coin: any): Promise<void>;
  getCoinSnapshots(symbol: string, days?: number): Promise<any[]>;
  getTopCoins(limit: number): Promise<any[]>;
  
  saveSentimentAnalysis(sentiment: any): Promise<void>;
  getSentimentAnalysis(symbol: string, date: string): Promise<any | null>;
  
  savePriceHistory(symbol: string, ohlcv: any): Promise<void>;
  getPriceHistory(symbol: string, days: number): Promise<any[]>;
  
  cleanupExpiredData(): Promise<void>;
}

class SQLiteStorageService implements StorageService {
  private db: Database.Database | null = null;
  private dbPath: string;

  constructor(dbPath: string = process.env.DATABASE_PATH || './sentiment_analyzer.db') {
    this.dbPath = dbPath;
  }

  async connect(): Promise<void> {
    try {
      // Create db directory if not exists
      const dir = path.dirname(this.dbPath);
      if (!fs.existsSync(dir)) {
        fs.mkdirSync(dir, { recursive: true });
      }

      // Open database
      this.db = new Database(this.dbPath);
      
      // Enable foreign keys
      this.db.pragma('foreign_keys = ON');
      
      // Create schema
      await this.initializeSchema();
      
      console.log(`✓ Connected to SQLite database: ${this.dbPath}`);
    } catch (error) {
      console.error('SQLite connection failed:', error);
      throw error;
    }
  }

  async disconnect(): Promise<void> {
    if (this.db) {
      this.db.close();
      console.log('✓ Disconnected from SQLite');
    }
  }

  private async initializeSchema(): Promise<void> {
    if (!this.db) throw new Error('Database not connected');

    const schema = `
      CREATE TABLE IF NOT EXISTS coins (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        symbol TEXT UNIQUE NOT NULL,
        name TEXT NOT NULL,
        price_usd REAL NOT NULL,
        market_cap_usd REAL,
        volume_24h_usd REAL,
        price_change_24h_percent REAL,
        price_change_7d_percent REAL,
        volatility_24h REAL,
        volatility_7d REAL,
        sentiment_score TEXT,
        sentiment_confidence REAL,
        sentiment_summary TEXT,
        market_rank INTEGER,
        trending_score INTEGER,
        headline_sample TEXT,
        timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
        source TEXT,
        updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
      );

      CREATE TABLE IF NOT EXISTS coin_snapshots (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        symbol TEXT NOT NULL,
        price_usd REAL NOT NULL,
        volume_24h_usd REAL,
        sentiment_score TEXT,
        sentiment_confidence REAL,
        timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
        FOREIGN KEY(symbol) REFERENCES coins(symbol)
      );

      CREATE TABLE IF NOT EXISTS sentiment_cache (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        symbol TEXT NOT NULL,
        analysis_date TEXT NOT NULL,
        sentiment_score TEXT NOT NULL,
        confidence REAL,
        summary TEXT,
        key_catalysts TEXT,
        risk_factors TEXT,
        short_term_outlook TEXT,
        volatility_warning BOOLEAN,
        generated_at DATETIME DEFAULT CURRENT_TIMESTAMP,
        model TEXT,
        tokens_used INTEGER,
        expires_at DATETIME,
        FOREIGN KEY(symbol) REFERENCES coins(symbol),
        UNIQUE(symbol, analysis_date)
      );

      CREATE TABLE IF NOT EXISTS price_history (
        id INTEGER PRIMARY KEY AUTOINCREMENT,
        symbol TEXT NOT NULL,
        date TEXT NOT NULL,
        open REAL,
        high REAL,
        low REAL,
        close REAL,
        volume REAL,
        timestamp DATETIME DEFAULT CURRENT_TIMESTAMP,
        FOREIGN KEY(symbol) REFERENCES coins(symbol),
        UNIQUE(symbol, date)
      );

      CREATE INDEX IF NOT EXISTS idx_coins_symbol ON coins(symbol);
      CREATE INDEX IF NOT EXISTS idx_coins_market_rank ON coins(market_rank);
      CREATE INDEX IF NOT EXISTS idx_snapshots_symbol_timestamp ON coin_snapshots(symbol, timestamp);
      CREATE INDEX IF NOT EXISTS idx_sentiment_symbol_date ON sentiment_cache(symbol, analysis_date);
      CREATE INDEX IF NOT EXISTS idx_sentiment_expires ON sentiment_cache(expires_at);
      CREATE INDEX IF NOT EXISTS idx_price_symbol_date ON price_history(symbol, date);
    `;

    // Split and execute each statement
    const statements = schema.split(';').filter(s => s.trim());
    for (const statement of statements) {
      this.db.exec(statement);
    }

    console.log('✓ Database schema initialized');
  }

  async saveCoinSnapshot(coin: any): Promise<void> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      INSERT OR REPLACE INTO coins (
        symbol, name, price_usd, market_cap_usd, volume_24h_usd,
        price_change_24h_percent, price_change_7d_percent,
        volatility_24h, volatility_7d, sentiment_score,
        sentiment_confidence, sentiment_summary, market_rank,
        trending_score, headline_sample, timestamp, source
      ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
    `);

    stmt.run(
      coin.symbol,
      coin.name,
      coin.price_usd,
      coin.market_cap_usd,
      coin.volume_24h_usd,
      coin.price_change_24h_percent,
      coin.price_change_7d_percent,
      coin.volatility_24h,
      coin.volatility_7d,
      coin.sentiment_score,
      coin.sentiment_confidence,
      coin.sentiment_summary,
      coin.market_rank,
      coin.trending_score,
      JSON.stringify(coin.headline_sample || []),
      new Date().toISOString(),
      coin.source
    );

    // Also save to snapshots table
    const snapshotStmt = this.db.prepare(`
      INSERT INTO coin_snapshots (symbol, price_usd, volume_24h_usd, sentiment_score, sentiment_confidence)
      VALUES (?, ?, ?, ?, ?)
    `);

    snapshotStmt.run(
      coin.symbol,
      coin.price_usd,
      coin.volume_24h_usd,
      coin.sentiment_score,
      coin.sentiment_confidence
    );
  }

  async getCoinSnapshots(symbol: string, days: number = 7): Promise<any[]> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      SELECT * FROM coin_snapshots
      WHERE symbol = ? AND timestamp > datetime('now', '-' || ? || ' days')
      ORDER BY timestamp DESC
    `);

    const results = stmt.all(symbol.toUpperCase(), days) as any[];
    return results;
  }

  async getTopCoins(limit: number = 50): Promise<any[]> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      SELECT * FROM coins
      ORDER BY market_rank ASC
      LIMIT ?
    `);

    const results = stmt.all(limit) as any[];
    
    // Parse JSON strings back to arrays
    return results.map(coin => ({
      ...coin,
      headline_sample: JSON.parse(coin.headline_sample || '[]'),
    }));
  }

  async saveSentimentAnalysis(sentiment: any): Promise<void> {
    if (!this.db) throw new Error('Database not connected');

    const expiresAt = new Date();
    expiresAt.setHours(expiresAt.getHours() + 24);

    const stmt = this.db.prepare(`
      INSERT OR REPLACE INTO sentiment_cache (
        symbol, analysis_date, sentiment_score, confidence,
        summary, key_catalysts, risk_factors, short_term_outlook,
        volatility_warning, generated_at, model, tokens_used, expires_at
      ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
    `);

    stmt.run(
      sentiment.symbol,
      sentiment.analysis_date,
      sentiment.sentiment_score,
      sentiment.confidence,
      sentiment.summary,
      JSON.stringify(sentiment.key_catalysts || []),
      JSON.stringify(sentiment.risk_factors || []),
      sentiment.short_term_outlook,
      sentiment.volatility_warning ? 1 : 0,
      new Date().toISOString(),
      sentiment.model,
      sentiment.tokens_used,
      expiresAt.toISOString()
    );
  }

  async getSentimentAnalysis(symbol: string, date: string): Promise<any | null> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      SELECT * FROM sentiment_cache
      WHERE symbol = ? AND analysis_date = ? AND expires_at > datetime('now')
    `);

    const result = stmt.get(symbol.toUpperCase(), date) as any;
    
    if (!result) return null;

    // Parse JSON strings
    return {
      ...result,
      key_catalysts: JSON.parse(result.key_catalysts || '[]'),
      risk_factors: JSON.parse(result.risk_factors || '[]'),
      volatility_warning: result.volatility_warning === 1,
    };
  }

  async savePriceHistory(symbol: string, ohlcv: any): Promise<void> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      INSERT OR REPLACE INTO price_history (
        symbol, date, open, high, low, close, volume
      ) VALUES (?, ?, ?, ?, ?, ?, ?)
    `);

    const today = new Date().toISOString().split('T')[0];
    stmt.run(
      symbol.toUpperCase(),
      today,
      ohlcv.open,
      ohlcv.high,
      ohlcv.low,
      ohlcv.close,
      ohlcv.volume
    );
  }

  async getPriceHistory(symbol: string, days: number = 7): Promise<any[]> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      SELECT * FROM price_history
      WHERE symbol = ? AND date > date('now', '-' || ? || ' days')
      ORDER BY date DESC
    `);

    const results = stmt.all(symbol.toUpperCase(), days) as any[];
    return results;
  }

  async cleanupExpiredData(): Promise<void> {
    if (!this.db) throw new Error('Database not connected');

    const stmt = this.db.prepare(`
      DELETE FROM sentiment_cache
      WHERE expires_at < datetime('now')
    `);

    const result = stmt.run();
    console.log(`✓ Cleaned up ${result.changes} expired sentiment records`);
  }
}

export default SQLiteStorageService;

USAGE IN EXPRESS APP
====================

File: backend/src/index.ts (updated)

import SQLiteStorageService from './storage';

// Initialize storage
const storage = new SQLiteStorageService();

// Connect at startup
app.listen(port, async () => {
  try {
    await storage.connect();
    
    // Run cleanup every hour
    setInterval(() => storage.cleanupExpiredData(), 60 * 60 * 1000);
    
    console.log(`✓ Server running on port ${port}`);
  } catch (error) {
    console.error('Failed to start server:', error);
    process.exit(1);
  }
});

// Graceful shutdown
process.on('SIGINT', async () => {
  await storage.disconnect();
  process.exit(0);
});

SQLITE vs MONGODB vs COSMOS DB
==============================

Feature              | SQLite     | MongoDB    | Cosmos DB
--------------------|------------|------------|----------
Local Dev            | ✅ File    | ✅ Local   | ❌ No
Cloud Hosting        | ❌ No      | ✅ Atlas   | ✅ Azure
Free Tier            | ∞          | ✅ 512MB   | ✅ 25GB
Setup Time           | 0 min      | 5 min      | 15 min
File Size            | < 100MB    | Unlimited  | Unlimited
Scalability          | Single DB  | Horizontal | Global
Replication          | ❌ No      | ✅ Yes     | ✅ Yes
Cloud Migration      | Manual     | Easy       | Easiest

RECOMMENDATION:

Start with: SQLite (simplest, zero setup)
Later move to: MongoDB Atlas (free cloud)
Production: Azure Cosmos DB (when deployed)

---

ADVANTAGES OF SQLITE
====================

✅ Zero external dependencies
✅ Single file database
✅ No server to run
✅ No credentials needed
✅ Works offline
✅ Fast for local development
✅ Easily shareable (single .db file)
✅ Great for testing
✅ < 100MB database size
✅ ACID transactions

LIMITATIONS
===========

❌ Not designed for horizontal scaling
❌ No built-in replication
❌ Single-writer limitation
❌ Not suitable for cloud apps (until you migrate)
❌ File-based (not ideal for network access)

MIGRATION STEPS (When Ready to Move to MongoDB)
==============================================

1. Install MongoDB locally or MongoDB Atlas

2. Create export script:
   node scripts/export-sqlite-to-mongodb.js

3. Update connection in backend:
   Change: const storage = new SQLiteStorageService();
   To:     const storage = new MongoDBStorageService();

4. Update .env:
   Remove: DATABASE_PATH
   Add:    MONGODB_URI=mongodb+srv://...

5. Test with: npm run dev

6. Database migrates automatically!

TROUBLESHOOTING
===============

"better-sqlite3 compilation failed"
- Install Python: https://www.python.org/downloads/
- Reinstall: npm install better-sqlite3

"Database locked"
- Close other connections to database
- Restart backend server

"No such table"
- Database schema not initialized
- Delete sentiment_analyzer.db and restart
- Schema auto-created on connect

"Data too large"
- SQLite works best < 100MB
- Consider migrating to MongoDB if > 100MB
- Run cleanup: storage.cleanupExpiredData()
