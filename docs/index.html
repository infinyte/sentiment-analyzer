Sentiment Analyzer

Real-time crypto sentiment, MARL tournaments, and social media intelligence.

View Repo Quick Start

What it does

Real-time cryptocurrency sentiment analysis platform combining market data, news, and social signals into Bull / Neutral / Bear outputs and trend reports.

“Results are displayed through an interactive React dashboard with MARL tournament tooling and social media intelligence.”

Key features

  • Live market data — CoinGecko OHLCV and price feeds
  • Social intelligence — 7-source scraper, sarcasm detection, FinBERT optional
  • MARL — Multi-agent tournaments, evolutionary orchestrator, paper/live broker adapters
  • Backtesting — Day-by-day simulation and equity curves

Quick Start

# Backend (localhost:3000)
cd backend
npm install
cp .env.example .env
npm run dev

# Frontend (localhost:5173)
cd frontend
npm install
npm run dev

API highlights

Core endpoints: /api/coins, /api/coins/:symbol, /api/sentiment/:symbol, /api/marl/competition/start.

Example: start a SINGLE tournament 
curl -s -X POST http://localhost:3000/api/marl/competition/start \
  -H "Content-Type: application/json" \
  -d '{ "mode":"SINGLE", "agents":[{"id":"alpha","riskProfile":"AGGRESSIVE"}], "symbols":["BTC"], "duration":200 }'
