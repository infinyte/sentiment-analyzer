CODE COMPATIBILITY
==================

All three options use the SAME backend code!

Class structure:
interface StorageService {
  async connect()
  async saveCoinSnapshot(coin)
  async getCoinSnapshots(symbol, days)
  async getTopCoins(limit)
  async saveSentimentAnalysis(sentiment)
  async getSentimentAnalysis(symbol, date)
  async savePriceHistory(symbol, ohlcv)
  async getPriceHistory(symbol, days)
}

Only differences:
1. Initial connection setup (different libraries)
2. Connection string format (.env variable)
3. That's it!

Example:
// SQLite
const storage = new SQLiteStorageService('./db.db');

// MongoDB
const storage = new MongoDBStorageService(process.env.MONGODB_URI);

// Cosmos DB (later)
const storage = new MongoDBStorageService(process.env.MONGODB_URI);
// Same code! Just different connection string!
