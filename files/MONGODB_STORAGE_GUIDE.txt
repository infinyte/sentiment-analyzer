SENTIMENT ANALYZER - MONGODB STORAGE IMPLEMENTATION
====================================================

OVERVIEW
--------

This guide shows how to use MongoDB for local development with smooth migration 
to Azure Cosmos DB for production.

KEY BENEFIT: MongoDB code runs identically on Azure Cosmos DB with zero changes!

---

INSTALLATION OPTIONS
---------------------

OPTION 1: LOCAL MONGODB (Recommended for Development)
======================================================

1. Install MongoDB Community Edition
   Windows: https://www.mongodb.com/try/download/community
   macOS: brew install mongodb-community
   Linux: sudo apt-get install -y mongodb-org

2. Start MongoDB locally
   Windows: mongod
   macOS: brew services start mongodb-community
   Linux: sudo systemctl start mongod

3. Verify it's running
   mongosh localhost:27017
   > show dbs

4. Set connection string in .env
   MONGODB_URI=mongodb://localhost:27017/sentiment_analyzer


OPTION 2: MONGODB ATLAS (Free Cloud, 512MB)
============================================

1. Sign up at https://www.mongodb.com/cloud/atlas
2. Create free tier cluster (512MB storage)
3. Create database user with password
4. Get connection string: mongodb+srv://user:pass@cluster.mongodb.net/?retryWrites=true
5. Add connection string to .env
   MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/sentiment_analyzer?retryWrites=true

Free Tier Limits:
- 512MB storage (enough for ~50K coin snapshots)
- 3 nodes (high availability)
- 100 concurrent connections
- Automatic backups

Best choice: MongoDB Atlas (no local setup needed, cloud-hosted)


OPTION 3: DOCKER MONGODB (For Everyone)
=======================================

1. Install Docker: https://www.docker.com/products/docker-desktop

2. Run MongoDB in Docker
   docker run -d \
     -p 27017:27017 \
     --name mongodb \
     -e MONGO_INITDB_ROOT_USERNAME=admin \
     -e MONGO_INITDB_ROOT_PASSWORD=password \
     mongo

3. Connection string
   MONGODB_URI=mongodb://admin:password@localhost:27017/sentiment_analyzer?authSource=admin

4. Stop when done
   docker stop mongodb
   docker rm mongodb

BACKEND IMPLEMENTATION
======================

File: backend/src/storage.ts

import { MongoClient, Db, Collection } from 'mongodb';

interface StorageService {
  connect(): Promise<void>;
  disconnect(): Promise<void>;
  
  saveCoinSnapshot(coin: any): Promise<void>;
  getCoinSnapshots(symbol: string, days?: number): Promise<any[]>;
  getTopCoins(limit: number): Promise<any[]>;
  
  saveSentimentAnalysis(sentiment: any): Promise<void>;
  getSentimentAnalysis(symbol: string, date: string): Promise<any | null>;
  
  savePriceHistory(symbol: string, ohlcv: any): Promise<void>;
  getPriceHistory(symbol: string, days: number): Promise<any[]>;
}

class MongoDBStorageService implements StorageService {
  private client: MongoClient;
  private db: Db | null = null;
  private coinsCollection: Collection | null = null;
  private sentimentCollection: Collection | null = null;
  private priceHistoryCollection: Collection | null = null;

  constructor(uri: string = process.env.MONGODB_URI || 'mongodb://localhost:27017/sentiment_analyzer') {
    this.client = new MongoClient(uri);
  }

  async connect(): Promise<void> {
    try {
      await this.client.connect();
      this.db = this.client.db('sentiment_analyzer');
      
      // Initialize collections
      this.coinsCollection = this.db.collection('coins');
      this.sentimentCollection = this.db.collection('sentiment_cache');
      this.priceHistoryCollection = this.db.collection('price_history');
      
      // Create indexes
      await this.createIndexes();
      
      console.log('✓ Connected to MongoDB');
    } catch (error) {
      console.error('MongoDB connection failed:', error);
      throw error;
    }
  }

  async disconnect(): Promise<void> {
    await this.client.close();
    console.log('✓ Disconnected from MongoDB');
  }

  private async createIndexes(): Promise<void> {
    // Coins indexes
    await this.coinsCollection?.createIndex({ symbol: 1 });
    await this.coinsCollection?.createIndex({ timestamp: -1 });
    await this.coinsCollection?.createIndex({ sentiment_score: 1, timestamp: -1 });
    await this.coinsCollection?.createIndex({ market_rank: 1 });
    
    // Sentiment indexes with TTL (auto-delete after 24 hours)
    await this.sentimentCollection?.createIndex(
      { generated_at: 1 },
      { expireAfterSeconds: 86400 }
    );
    await this.sentimentCollection?.createIndex({ symbol: 1, analysis_date: -1 });
    
    // Price history indexes
    await this.priceHistoryCollection?.createIndex({ symbol: 1, date: -1 });
    await this.priceHistoryCollection?.createIndex({ date: -1 });
    
    console.log('✓ Database indexes created');
  }

  async saveCoinSnapshot(coin: any): Promise<void> {
    if (!this.coinsCollection) throw new Error('Not connected to database');
    
    const document = {
      ...coin,
      timestamp: new Date(),
    };
    
    // Update or insert (upsert)
    await this.coinsCollection.updateOne(
      { symbol: coin.symbol, date: new Date().toISOString().split('T')[0] },
      { $set: document },
      { upsert: true }
    );
  }

  async getCoinSnapshots(symbol: string, days: number = 7): Promise<any[]> {
    if (!this.coinsCollection) throw new Error('Not connected to database');
    
    const fromDate = new Date();
    fromDate.setDate(fromDate.getDate() - days);
    
    const snapshots = await this.coinsCollection
      .find({
        symbol: symbol.toUpperCase(),
        timestamp: { $gte: fromDate },
      })
      .sort({ timestamp: -1 })
      .toArray();
    
    return snapshots;
  }

  async getTopCoins(limit: number = 50): Promise<any[]> {
    if (!this.coinsCollection) throw new Error('Not connected to database');
    
    // Get latest snapshot for each coin
    const coins = await this.coinsCollection
      .aggregate([
        { $sort: { timestamp: -1 } },
        { $group: {
            _id: '$symbol',
            doc: { $first: '$$ROOT' }
          }
        },
        { $replaceRoot: { newRoot: '$doc' } },
        { $sort: { market_rank: 1 } },
        { $limit: limit }
      ])
      .toArray();
    
    return coins;
  }

  async saveSentimentAnalysis(sentiment: any): Promise<void> {
    if (!this.sentimentCollection) throw new Error('Not connected to database');
    
    const document = {
      ...sentiment,
      generated_at: new Date(),
    };
    
    await this.sentimentCollection.updateOne(
      { symbol: sentiment.symbol, analysis_date: sentiment.analysis_date },
      { $set: document },
      { upsert: true }
    );
  }

  async getSentimentAnalysis(symbol: string, date: string): Promise<any | null> {
    if (!this.sentimentCollection) throw new Error('Not connected to database');
    
    const sentiment = await this.sentimentCollection.findOne({
      symbol: symbol.toUpperCase(),
      analysis_date: date,
    });
    
    return sentiment;
  }

  async savePriceHistory(symbol: string, ohlcv: any): Promise<void> {
    if (!this.priceHistoryCollection) throw new Error('Not connected to database');
    
    const document = {
      symbol: symbol.toUpperCase(),
      date: new Date().toISOString().split('T')[0],
      ...ohlcv,
    };
    
    await this.priceHistoryCollection.updateOne(
      { symbol: symbol.toUpperCase(), date: document.date },
      { $set: document },
      { upsert: true }
    );
  }

  async getPriceHistory(symbol: string, days: number = 7): Promise<any[]> {
    if (!this.priceHistoryCollection) throw new Error('Not connected to database');
    
    const fromDate = new Date();
    fromDate.setDate(fromDate.getDate() - days);
    
    const history = await this.priceHistoryCollection
      .find({
        symbol: symbol.toUpperCase(),
        date: {
          $gte: fromDate.toISOString().split('T')[0],
        },
      })
      .sort({ date: -1 })
      .toArray();
    
    return history;
  }
}

export default MongoDBStorageService;


USAGE IN EXPRESS APP
====================

File: backend/src/index.ts (updated)

import MongoDBStorageService from './storage';

// Initialize storage
const storage = new MongoDBStorageService();

// Connect at startup
app.listen(port, async () => {
  try {
    await storage.connect();
    console.log(`✓ Server running on port ${port}`);
  } catch (error) {
    console.error('Failed to start server:', error);
    process.exit(1);
  }
});

// Use in routes
app.get('/api/coins', async (req, res) => {
  try {
    const coins = await storage.getTopCoins(50);
    res.json({ data: coins });
  } catch (error) {
    res.status(500).json({ error: 'Failed to fetch coins' });
  }
});

app.get('/api/coins/:symbol', async (req, res) => {
  try {
    const { symbol } = req.params;
    const snapshots = await storage.getCoinSnapshots(symbol, 7);
    res.json({ coin: snapshots[0], history: snapshots });
  } catch (error) {
    res.status(500).json({ error: 'Failed to fetch coin' });
  }
});

// Graceful shutdown
process.on('SIGINT', async () => {
  await storage.disconnect();
  process.exit(0);
});

---

MIGRATION TO AZURE COSMOS DB (Later)
=====================================

When you have Azure access, switch with ONE line change:

Before (Local):
MONGODB_URI=mongodb://localhost:27017/sentiment_analyzer

After (Azure Cosmos DB):
MONGODB_URI=mongodb+srv://<cosmosdb>:<password>@<cosmosdb>.mongo.cosmos.azure.com:10255/?retryWrites=true&replicaSet=globaldb

Everything else works identically!

TROUBLESHOOTING
===============

"Cannot connect to MongoDB"
- Verify MongoDB is running
- Check connection string in .env
- Try: mongosh to test locally

"Authentication failed"
- Check username/password
- Ensure @ symbols are URL-encoded (%40)
- Verify IP whitelist (MongoDB Atlas)

"No collections found"
- Collections are created automatically on first insert
- Indexes are created on first connect

"Data disappears after 24 hours"
- That's the TTL index working correctly!
- Sentiment data intentionally expires
- Coin snapshots are permanent
