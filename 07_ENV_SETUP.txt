BACKEND ENVIRONMENT VARIABLES (.env file)
==========================================

Copy this to backend/.env and fill in your values.

NODE_ENV=development
PORT=3000
FRONTEND_URL=http://localhost:5173

# Get from https://newsapi.org (free tier)
NEWSAPI_API_KEY=your_newsapi_key_here

# Get from https://console.anthropic.com
CLAUDE_API_KEY=your_claude_key_here

# CORS settings
ALLOWED_ORIGINS=http://localhost:5173

# For API authentication
API_SECRET_KEY=dev-secret-key-for-admin-endpoints


FRONTEND ENVIRONMENT VARIABLES (.env file)
===========================================

Copy this to frontend/.env and fill in your values.

VITE_API_BASE_URL=http://localhost:3000
VITE_APP_NAME=Sentiment Analyzer


GETTING API KEYS
================

1. NewsAPI (Free Tier)
   - Go to: https://newsapi.org
   - Click "Get API Key"
   - Create free account
   - Copy API key
   - Limit: 500 requests per day
   - Cost: Free for development

2. Claude API (Anthropic)
   - Go to: https://console.anthropic.com
   - Sign in or create account
   - Go to API Keys
   - Create new API key
   - Copy and save securely
   - Cost: Pay-per-use (~$0.01-0.03 per request)
   - Estimated: $8-15/month for daily sentiment analysis

3. CoinGecko API (Free)
   - No API key needed
   - Free public API
   - Rate limit: Check documentation
   - Cost: Free

4. Azure Storage (For Production)
   - Create Azure account
   - Create Storage Account
   - Copy connection string
   - Cost: Free tier (5GB)

- No authentication needed for local testing

PRODUCTION (Azure)
- NODE_ENV=production
- FRONTEND_URL=https://sentiment-dashboard-app.azurewebsites.net
- ALLOWED_ORIGINS=https://sentiment-dashboard-app.azurewebsites.net
- API running on: https://sentiment-api-app.azurewebsites.net
- Authentication required for admin endpoints


SECURING SECRETS
================

DO NOT commit .env files to Git!

1. Add to .gitignore:
   .env
   .env.local
   *.env

2. For Azure deployment:
   - Use GitHub Secrets
   - Use Azure Key Vault
   - Never hardcode keys

3. For local development:
   - Copy .env.example to .env
   - Edit .env with your values
   - Keep .env files local only


TROUBLESHOOTING
===============

Missing API key error:
- Verify .env file exists in backend/ directory
- Verify all required variables are set
- Check spelling of variable names (case-sensitive)
- Restart backend server after changing .env

401 Unauthorized errors:
- Check API keys are valid
- Check API key format (should not have extra spaces)
- Verify key hasn't expired or been revoked
- Try requesting new key from API provider

CORS errors:
- Check ALLOWED_ORIGINS matches frontend URL
- Verify CORS is enabled on backend
- Check browser console for specific error

Network timeout:
- Check internet connection
- Verify API endpoints are accessible
- Check rate limits (NewsAPI, CoinGecko)
- Add delay between requests if rate limited
